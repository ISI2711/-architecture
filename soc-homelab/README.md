# ISI2711 Homelab Architecture

This README provides an overview of the ISI2711 Homelab Architecture project.

# Lab Overview

This SOC home lab simulates a real-world security operations environment.

## Components
- pfSense (Firewall/Router)
- Ubuntu Server (Infrastructure)
- Security Onion (SIEM)
- Kali Linux (Attacker machine)
- Nessus (Vulnerability Scanner)

## Objective
To monitor, detect, and respond to security events.
